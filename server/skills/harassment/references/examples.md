# Example Cases

## Dismiss

**Case:** Score 0.6, heated exchange between two users in comments about a film opinion
- Comments: both parties gave as good as they got, the argument concluded naturally
- Reporter: the losing party in the argument
- User history: no prior reports
- Decision: `dismiss_report`
- Reasoning: Mutual heated argument, not one-sided targeting. Both parties participated equally. Dispute appears resolved. This is not harassment.

## Warn

**Case:** Score 0.45, post publicly calls out a specific user by username for "always ruining discussions"
- Comments: a few users pile on, a few defend the target
- User history: similar callout post targeting the same user 6 weeks ago
- Decision: `warn_user`
- Reasoning: Targeting a specific user publicly, second instance. Pattern is forming but not yet sustained. Warning is appropriate before escalation.

## Remove

**Case:** Score 0.7, third post in one week referencing and mocking the same user
- Comments: growing pile-on, target has not responded (possibly avoiding the platform)
- User history: 4 reports in the past month, 3 of them involving the same target
- Decision: `remove_post`
- Reasoning: Sustained targeting of a specific user. Pattern is clear — 3 posts this month directed at the same person. Community is amplifying the harassment. Removal is justified.
